# evernote-tools
My evernote tools

## Setup
```
$ git clone git@github.com:tayutaedomo/evernote-tools.git
$ cd evernote-tools
$ source venv/bin/activate
$ pip install -r requirements.txt
```

## Scripts
```
$ export EVERNOTE_DEV_TOKEN="<Your developer token for Sandbox>"
$ python scripts/get_notes.py
```
